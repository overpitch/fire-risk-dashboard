Troubleshooting Tips
If you're seeing 401 Unauthorized errors:
  • Check if your API key is valid and has the correct permissions
  • Verify that the token is being correctly passed in the request
  • Check if the token has expired (they typically have a short lifespan)
  • Verify that you have access to the specific stations you're requesting
  • Check if there are any rate limits or usage restrictions on your account
If the API is intermittently failing:
  • The API service might be experiencing temporary issues
  • There might be network connectivity problems
  • You might be hitting rate limits during peak usage times
  • The token might be expiring before you use it in some cases
